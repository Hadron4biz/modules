Currency rate update extension

Note that this module works only with National Bank of Poland service!

This module depends on 'Currency rate update' module. It adds the 'Table name' and 'Table rate date' column.

Configuration

1.The base currency must be PLN.

2.And its base rate must be 1.

3.Remember also that this module works only if you choose the National Bank of Poland.

Rozszerzenie modułu do aktualizacji kursu walut

Uwaga ten moduł współpracuje wyłącznie z serwisem Narodowego Banku Polskiego!

Moduł ten zależny jest od modułu 'Currency rate update'. Dodaje on kolumny "Nazwa tabeli" oraz "Data kursu".

Konfiguracja

1.Bazową walutą musi być PLN.

2.Domyślna wartość musi wynosić 1,00.

3.Pamiętaj aby wybrać NBP jako serwer wykorzystywany do aktualizacji kursu.
